 ## Summary
  Twitch Friends is a Chrome extension for Twitch that shows which streamers your friends are watching. We collect
  only the data needed to provide this functionality.

  ## Data We Collect
  1) **Twitch account data**
  - Twitch ID, login, display name, avatar.
  - Email may be available via OAuth but is not stored.

  2) **Social data**
  - Friends list and friend requests.

  3) **Viewing status**
  - The channel login/name you are currently watching and the last update time.

  4) **Local browser data**
  - Session token and profile for login, plus cached status.

  5) **Technical data**
  - Service logs (e.g., IP, user-agent) may be stored by infrastructure providers.

  ## How We Use Data
  - Authenticate users via Twitch.
  - Show friends and their viewing statuses.
  - Ensure service reliability and prevent abuse.

  ## Data Sharing
  - With your friends (only after a request is accepted): your viewing status and public profile data.
  - With infrastructure providers (Supabase, Twitch) to operate the service.
  - We do **not** sell data or share it for advertising.

  ## Data Retention & Deletion
  - Data is stored while you use the service.
  - Viewing status is updated periodically and may expire when stale.
  - You can request deletion by contacting us at civinfinityru@gmail.com.

  ## Security
  - Data is stored in Supabase using standard security controls and access policies.
  - Secrets (e.g., Twitch Client Secret) are never stored in the extension.

  ## Your Choices
  - You can revoke access in Twitch (Connections).
  - You can uninstall the extension and request data deletion.

  ## Children
  This product is not intended for users under 13 years old.

  ## Changes
  We may update this policy. The latest version will always be available at this URL.
